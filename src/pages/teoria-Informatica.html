AULA #1

Como Ligar o Computador

Apresentação de dois modelos: Desktop (tradicional) e Notebook.

No Desktop, ligar o gabinete e o monitor através do botão "on/off".

No Notebook, localizar e pressionar o botão "on/off" para iniciar.

Processo de Inicialização:

Equipamentos antigos como televisões respondem automaticamente, mas no computador ocorre um processo de inicialização.

O sistema é carregado durante esse processo.

Tela Inicial do Windows:

Após a inicialização, o sistema apresenta uma tela inicial.

Descrição da tela, incluindo relógio no canto inferior direito, barra de tarefas com principais recursos e logotipo do Windows.

Significado do Sistema:

Analogia: O sistema é como o ambiente de trabalho na cozinha de uma casa.

Necessidade do sistema para utilizar as ferramentas (programas) no computador.

Resumo da Aula:

1. Ligar o computador pelo botão "on/off".

2. No Desktop, ligar também o monitor.

3. No Notebook, localizar o botão "on/off".

4. Aguardar o sistema iniciar.

5. Tela inicial do Windows: relógio, barra de tarefas, logotipo.

6. Sistema como ambiente de trabalho; ferramentas são os programas.

AULA #2

Como Utilizar o Mouse

Boas-vindas à segunda aula do Curso de Informática.

Foco na aprendizagem inicial do uso do mouse.

Uso do Mouse

Apresentação de diferentes modelos de mouse: sem fio, com fio e de notebook.

Função básica do mouse: movimentar a seta na tela.

Demonstração prática de como usar o mouse no notebook e no computador tradicional.

Funções dos Botões do Mouse

Botão direito: Apresentação de opções contextuais ao clicar em diferentes áreas.

Botão esquerdo: Utilização para abrir, fechar e escolher opções.

Desligamento do Computador

Explicação sobre como desligar o computador.

Demonstração prática do processo, destacando o botão "Iniciar" e a opção "Desligar".
